## WTG.Query.RAG

This project implements a Retrieval-Augmented Generation (RAG) pipeline using LangChain.
It can ingest PDF, Markdown, and plain text documents, index them into a vector store,
and answer questions using an LLM with retrieved context.

## Key Features

- PDF document loading via PyPDF
- Markdown and plain text support
- Chroma vector store for similarity search
- OpenAI embeddings and chat model
- Interactive Q&A loop
